Athens Through Time - Vampire: The Masquerade LARP

"In the city of shadows, every thread is pulled by a hand unseen."

An immersive web experience for the Athens Through Time V5 Chronicle, built with Vue 3 and Tailwind CSS. This website serves as the digital portal for players to explore the lore, setting, and gallery of the LARP.
